# KEITO_DAISENSO

ブラウザで動く、1ステージだけのミニタワーディフェンスゲームです。

Codexと一緒に、小さな `game.html` から育てていく開発ログ用のリポジトリです。

## 遊び方

1. `game.html` をブラウザで開きます。
2. お金を使ってユニットを召喚します。
3. 味方と敵は自動で進み、自動で戦います。
4. 敵を倒すとお金が増えます。
5. 敵拠点を先に壊すと勝利です。

## 初回版に入っているもの

- 横スクロール風の1画面バトル
- ユニット召喚
- 敵の自動出現
- 自動戦闘
- お金システム
- 敵撃破報酬
- 勝利、敗北、リスタート
- 外部通信なしの安全な静的HTML

## 方針

このリポジトリは、既存ゲームの素材やコードを使わず、HTML/CSS/JavaScriptで作る非公式ファン制作です。

初回版は `game.html` だけで完結させ、あとから必要に応じてファイル分割やステージ追加を検討します。
