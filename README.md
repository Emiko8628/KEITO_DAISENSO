# KEITO_DAISENSO

ブラウザで動く、1ステージだけのミニタワーディフェンスゲームです。

Codexと一緒に、小さな `game.html` から育てていく開発ログ用のリポジトリです。

## 遊び方

1. `game.html` をブラウザで開きます。
2. お金を使ってユニットを召喚します。
3. 右の味方拠点からユニットが出て、左の敵拠点へ進みます。
4. 敵を倒したり敵拠点を攻撃したりすると、お金と経験値が増えます。
5. 左の敵拠点のHPを0にするとステージクリアです。

## 現在のステージ

- 編: `大地編`
- ステージ: `大地をゆるがすワンワンステージ`
- クリア目標: 敵拠点HPを `0` にする
- 経験値通知: 経験値 `100` で「経験値がたまったよ！」を表示
- クリアボーナス: お金 `+120`

経験値は保存されません。今は1プレイごとの進行通知として動き、クリア条件ではありません。

- 横スクロール風の1画面バトル
- ユニット召喚
- 敵の自動出現

- お金システム

- 召喚クールタイム
- 攻撃ヒット演出
- ステージ開始、クリア演出

- Google Analytics側の処理は、Googleのプライバシーポリシーとデータ処理規約に従います

## 味方キャラクター設定

- まるねこ: 基本キャラ。低コストで呼びやすく、HP・攻撃・移動速度のバランスが良い最初の主役です。
- かたいねこ: 守りの壁キャラ。攻撃は弱めですがHPが高く、前線を止めて味方を守ります。
- こうげきねこ: 攻撃キャラ。コストは高めですが攻撃力が高く、敵拠点を早く壊したい時に活躍します。

最初のステージでは、`まるねこ 50`、`かたいねこ 80`、`こうげきねこ 110` の3ボタンで召喚できます。

召喚クールタイムは、`まるねこ 1.4秒`、`かたいねこ 2.6秒`、`こうげきねこ 3.2秒` です。

## 敵キャラクター設定

- ぴょこネコ: すばやい通常敵。HPは低めで、最初のステージによく出ます。
- にょろゴースト: ゆっくりだけど硬い敵。倒すとお金が多めにもらえます。
- わちゃわちゃトリオ: たまに出るにぎやか敵。出現頻度は低めで、ステージのアクセントです。

最初のステージでは、`ぴょこネコ 70%`、`にょろゴースト 20%`、`わちゃわちゃトリオ 10%` くらいのバランスで出ます。

経験値は、`ぴょこネコ +18`、`にょろゴースト +30`、`わちゃわちゃトリオ +26` を基本に、敵拠点への攻撃でも少し増えます。
経験値が満タンになってもバトルは続きます。ステージクリアは、左の敵拠点を壊した時です。
